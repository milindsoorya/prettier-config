# Custom prettier-config

# Prettier configuration
```
{
  "semi": true,
  "singleQuote": true,
  "tabWidth": 2,
  "trailingComma": "none",
  "printWidth": 100
}

```
# Usage

## Install the package
```
npm i @milindsoorya/prettier-config --save-dev

```
## use it in the project 
```
{
  "name": "projectname",
  "version": "0.1.0",
  "prettier": "@milindsoorya/prettier-config",
}
```
